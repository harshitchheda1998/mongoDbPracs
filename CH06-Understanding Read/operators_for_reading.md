# Operators used in Reading data
* Query Selectors
	* Used to locate data
	* They don't change the data
	* Types
		* Comparision
		* Logical
		* Element
		* Evaluation
		* Array
		* Comments
		* Geospatial

* Projection Operators
	* Used to change the representation of data on result
	* They don't change the data
	* Types
		* $ - Projection 
		* $elemMatch
		* $meta
		* $slice
